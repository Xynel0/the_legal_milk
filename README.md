# 🥛 Legal Milk

> Um repositório de aprendizado em Ciência da Computação.

## Sobre

O **Legal Milk** é um projeto educacional que tem como objetivo reunir materiais de estudo de Ciência da Computação em um único lugar, de forma organizada, gratuita e de fácil acesso.

A proposta é servir como um ponto de apoio para estudantes da graduação, vestibulandos, autodidatas e qualquer pessoa interessada na área.

O projeto ainda está em seus estágios iniciais, mas a ideia é construir uma plataforma que concentre teoria, exercícios e soluções para as principais disciplinas de um curso de Ciência da Computação.

---

## Objetivos

* 📚 Centralizar materiais de estudo.
* 🧠 Facilitar a revisão de conteúdos.
* 💻 Disponibilizar listas de exercícios e soluções comentadas.
* 🤝 Incentivar a colaboração entre estudantes.
* 🌎 Tornar o conhecimento mais acessível.

---

## Estrutura planejada

A organização dos conteúdos seguirá uma estrutura semelhante a esta:

```text
Semestre
│
├── Disciplina
│   ├── Conteúdo
│   │   ├── Teoria
│   │   ├── Exercícios
│   │   ├── Soluções
│   │   └── Referências
│   └── ...
└── ...
```


## Status do projeto

🚧 **Em desenvolvimento**

Atualmente estamos desenvolvendo a estrutura inicial do site. Os conteúdos serão adicionados gradualmente.

---

## Nossa visão

Queremos construir uma biblioteca de conhecimento organizada, colaborativa e duradoura, onde qualquer estudante possa encontrar rapidamente o conteúdo de que precisa para aprender Ciência da Computação.

Se o projeto puder ajudar alguém a compreender um conceito, resolver um exercício ou despertar interesse pela computação, ele já terá cumprido seu propósito.
